# RikTech AI - Full Version (Starter)
This project is a visual AI Chatbot Builder scaffold using Next.js (App router), TypeScript, Tailwind CSS, Zustand, lucide-react, framer-motion and react-draggable.

This archive is a working starter. To run locally:
1. npm install
2. Create .env.local with OPENAI_API_KEY
3. npm run dev

See instructions in the next message for step-by-step OpenAI API setup and Vercel deployment.
